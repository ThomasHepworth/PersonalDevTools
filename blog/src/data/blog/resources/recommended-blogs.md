A collection of engineering blogs I follow regularly. These range from individual practitioners sharing deep technical insights to company engineering blogs documenting how they solve problems at scale.

## Software Engineering Blogs

### Individual Blogs

| Blog | Description |
| --- | --- |
| [**Paul Graham**](https://paulgraham.com/) | Essays on programming, startups, and technology from the co-founder of Y Combinator. |
| [**Martin Fowler**](https://martinfowler.com/) | Articles on software development, architecture, and design patterns from one of the most influential voices in the industry. |
| [**Robin Linacre**](https://www.robinlinacre.com/) | Writes about data linkage, engineering, data science, and random musings. Great for getting an overview of Splink and record linkage. |

## Data Engineering Blogs

### Company Engineering Blogs

| Blog | Description |
| --- | --- |
| [**Uber Engineering**](https://www.uber.com/en-GB/blog/engineering/) | Deep dives into data engineering, machine learning, and software development at massive scale. |
| [**Airbnb Engineering**](https://medium.com/airbnb-engineering) | Covers data engineering, machine learning, and the systems that power Airbnb. |
| [**Netflix Tech Blog**](https://netflixtechblog.com/) | Insights into the engineering challenges of streaming at scale, including data pipelines, ML, and infrastructure. |
| [**Spotify Engineering**](https://engineering.atspotify.com/) | Articles on data engineering, machine learning, and building personalized experiences. |
| [**Ministry of Justice**](https://ministryofjustice.github.io/data-and-analytics-engineering/blog/) | Data and analytics engineering in the UK public sector. |
| [**LinkedIn Engineering**](https://engineering.linkedin.com/blog) | Technical posts on data engineering, machine learning, and software development at LinkedIn. |
